# TS Pizza App Cliche

Простой демонстрационный проект на TypeScript: меню пиццы, заказы и операции с ними.

Как запустить локально:

1. Установить зависимости:
```
npm install
```

2. Скомпилировать TypeScript в `dist`:
```
npm run build
```

3. Открыть `index.html` через статический сервер (например Live Server в VS Code) или раздавать `dist`.

Файлы, которые не должны попадать в репозиторий указаны в `.gitignore`.
